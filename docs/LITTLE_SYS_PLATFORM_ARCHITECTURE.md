# Little Sys — Future Platform Architecture

> Future vision document.

## Purpose

Little Sys is the shared platform layer powering all Little-* products.

## Core idea

Make it easy to build new domain systems with:
- shared primitives
- shared patterns
- shared learning model

## Core subsystems (future)
- identity and permissions
- event and immutable history
- rules engine
- REPL and execution engine
- MCP integration
- AI orchestration layer
- observability
- local-first sync
- UI design system

Future vision only.
