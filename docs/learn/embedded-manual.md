# Embedded Manual (User Brain View)

This is a **reframed version** of the embedded manual.

Focus:
- How to think
- Not just how to use

## Core Idea

You are not operating a dashboard.

You are operating:

- A system
- A feedback loop
- A decision engine

## The 4 Surfaces

1. Observe (Dashboard)
2. Understand (Learn)
3. Inspect (REPL)
4. Reason (Copilot)

## Loop

Observe → Hypothesize → Inspect → Adjust → Repeat

## Golden Rule

> Never act before inspecting.
