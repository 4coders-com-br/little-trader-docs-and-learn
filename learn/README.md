# 🧠 Learn — The User's Brain

> This section is NOT product documentation.
>
> This is the **cognitive layer** of Little Trader:
> how to think, learn, reason, and operate using the system.

---

## What this is

The `/learn` folder contains:

- Mental models
- Educational content
- System understanding
- Developer intuition
- Strategy thinking frameworks

It is the **"brain of the user"**, not the system itself.

---

## What this is NOT

This is NOT:

- Feature specification
- Infrastructure documentation
- DevOps instructions
- API contracts

Those live in `/docs` and root-level product docs.

---

## Core Concept

Little Trader has 2 layers:

### 1. Product Layer (docs/)
- What the system does
- How it runs
- How to deploy

### 2. Cognitive Layer (learn/)
- How to think about trading
- How to reason about the system
- How to use REPL, AI, and rules effectively
