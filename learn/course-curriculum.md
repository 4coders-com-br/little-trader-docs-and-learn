# Course Curriculum (Brain Layer)

This file reframes curriculum as a **human development path** instead of system documentation.

## Why it belongs here

The course is not a product feature spec.
It is a way to shape the user's mental model.

## What this teaches

- How to understand the codebase
- How to reason about trading systems
- How to use REPL, copilot, and strategy tools responsibly
- How to grow from operator to builder

## Suggested path

1. Orientation
2. REPL literacy
3. Architecture literacy
4. Rules and strategy thinking
5. Data intuition
6. Operational confidence

## Outcome

The goal is not only to use Little Trader.

The goal is to become the kind of person who can:
- inspect it
- question it
- improve it
- trust it carefully

## Relationship to product docs

- `/docs` explains the machine
- `/learn` develops the human
